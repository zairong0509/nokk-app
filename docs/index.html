🛡️ NOKK - 홈 안전 음성 앱

개인정보 처리방침 서비스 이용약관

집에 혼자 있을 때 당신을 지켜주는 안전 음성

NOKK는 집에 혼자 있는 여성이나 아이들이 위험한 상황에서 남자 목소리를 재생하여 안전을 지키는 앱입니다.

주요 기능

  • 🎤 다양한 상황별 남자 음성 (배달, 야간 방문, 위협 상황, 낯선 방문자)
  • 🎵 3가지 톤 선택 (일반, 단호, 화난)
  • 🌍 4개 언어 지원 (한국어, 영어, 일본어, 스페인어)
  • ⚡ 빠른 실행 버튼
  • 🎨 다크 모드 지원
  • 🔒 개인정보 수집 없음

사용 시나리오

  • 📦 배달원 응대: "문 앞에 놔주세요", "고맙습니다"
  • 🌙 야간 방문자: "너무 늦었어요", "내일 오세요", "자고 있어요"
  • ⚠️ 위협 상황: "경찰 부르고 있어요", "카메라 있어요", "나가세요"
  • ❓ 낯선 방문자: "누구세요?", "바빠요", "관심 없어요"
