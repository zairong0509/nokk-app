- **최대**: 8개

## 🎯 촬영할 화면 (우선순위)

### 1. HomeScreen (필수) ⭐⭐⭐
- **내용**: 메인 화면, 카테고리 목록
- **강조점**: 깔끔한 UI, 다크 모드
- **설명**: "다양한 안전 사운드를 한눈에"

### 2. ToneSelectScreen (필수) ⭐⭐⭐
- **내용**: 톤 선택 화면 (밝은, 차분한, 강한)
- **강조점**: 3가지 톤 옵션
- **설명**: "상황에 맞는 톤 선택"

### 3. LanguageSelectScreen (필수) ⭐⭐
- **내용**: 언어 선택 (한국어, English, 日本語, 中文)
- **강조점**: 4개 언어 지원
- **설명**: "글로벌 언어 지원"

### 4. SettingsScreen (권장) ⭐⭐
- **내용**: 설정 화면, 다크모드 토글
- **강조점**: 간단한 설정
- **설명**: "간편한 설정"

### 5. SearchScreen (선택) ⭐
- **내용**: 검색 기능
- **설명**: "원하는 사운드를 빠르게 검색"

## 📸 스크린샷 촬영 방법

### Android 에뮬레이터 사용
```bash
cd Desktop/Nokk
npx expo start
# 그 후 'a' 키를 눌러 Android 에뮬레이터 실행
```

### 스크린샷 캡처
1. 에뮬레이터에서 각 화면으로 이동
2. Windows: `Win + Shift + S` 또는 에뮬레이터의 카메라 버튼
3. 파일을 `store/screenshots/android/` 폴더에 저장

### 파일명 규칙
- `01-home-screen.png`
- `02-tone-select.png`
- `03-language-select.png`
- `04-settings.png`
- `05-search.png`

1. **다크 모드로 촬영**: 앱이 다크 모드를 지원하므로, 다크 모드로 촬영하면 더 멋있어 보임
