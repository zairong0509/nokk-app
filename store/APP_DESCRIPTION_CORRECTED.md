### NOKK는:
**집에 혼자 있는 여성이나 아이들이 위험한 상황에서 남자 목소리를 재생하여 안전을 지키는 홈 안전 음성 앱**

### 사용 시나리오:
1. 📦 **배달원 응대**: "문 앞에 놔주세요", "고맙습니다"
2. 🌙 **야간 방문자**: "너무 늦었어요", "내일 오세요", "자고 있어요"
3. ⚠️ **위협 상황**: "경찰 부르고 있어요", "카메라 있어요", "나가세요"
4. ❓ **낯선 방문자**: "누구세요?", "바빠요", "관심 없어요"

## 🎤 주요 기능

1. **상황별 음성**
   - 배달 (Delivery): 5개 문구
   - 낯선 방문자 (Unknown Visitor): 6개 문구
   - 위협 상황 (Threat): 6개 문구
   - 야간 방문 (Night): 4개 문구
   - 일반 응답 (General): 4개 문구

2. **3가지 톤**
   - Normal (일반): 차분하고 자신감 있는
   - Firm (단호): 단호하고 직접적인
   - Angry (화난): 위협적인 (프리미엄)

3. **4개 언어**
   - 한국어 (ko)
   - English (en)
   - 日本語 (ja)
   - Español (es)

4. **빠른 실행**
   - 무료: 3개 버튼
   - 프리미엄: 5개 버튼 (커스터마이징 가능)

1. **혼자 사는 여성**
   - 배달 받을 때
   - 낯선 방문자 응대 시
   - 야간 방문자 거부 시
