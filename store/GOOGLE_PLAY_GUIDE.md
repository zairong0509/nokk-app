## 🔐 1단계: Google Play Console 계정 생성

### 1.1 개발자 계정 등록
1. https://play.google.com/console 접속
2. "시작하기" 클릭
3. Google 계정으로 로그인
4. 개발자 등록 양식 작성:
   - **개발자 이름**: NOKK 또는 본인 이름
   - **이메일**: support@nokk.app 또는 본인 이메일
   - **웹사이트**: https://nokk.app (또는 GitHub Pages URL)
5. **등록비 $25 결제** (일회성, 평생 사용)
6. 약관 동의 및 제출

### 1.2 계정 승인 대기
- 보통 24-48시간 소요
- 이메일로 승인 알림 수신

## 📱 2단계: 앱 생성

### 2.1 새 앱 만들기
1. Console 홈페이지에서 "앱 만들기" 클릭
2. 앱 세부정보 입력:
   - **앱 이름**: NOKK
   - **기본 언어**: 한국어
   - **앱 또는 게임**: 앱
   - **무료 또는 유료**: 무료
3. 선언문 체크박스 선택:
   - ✅ Google Play 개발자 프로그램 정책 준수
   - ✅ 미국 수출법 준수
4. "앱 만들기" 클릭

## 🔒 4단계: 앱 콘텐츠

### 4.1 콘텐츠 등급
**경로**: 앱 콘텐츠 > 콘텐츠 등급

1. "설문지 시작" 클릭
2. 이메일 주소 입력: support@nokk.app
3. 카테고리 선택: **유틸리티, 생산성, 통신 또는 기타**
4. 설문지 작성:

**폭력성**
- 게임에 폭력적인 장면이 포함되어 있나요? → 아니요

**성적 콘텐츠**
- 성적인 주제가 포함되어 있나요? → 아니요

**언어**
- 욕설이나 비속어가 있나요? → 아니요

**약물, 담배, 주류**
- 관련 콘텐츠가 있나요? → 아니요

**도박**
- 도박 관련 콘텐츠가 있나요? → 아니요

5. 등급 계산 → **모든 연령** 또는 **12+**

### 4.5 데이터 보안
**경로**: 앱 콘텐츠 > 데이터 보안

**매우 중요!** NOKK는 데이터를 수집하지 않으므로:

1. "시작하기" 클릭
2. **데이터 수집 및 공유**:
   - "앱에서 필수 사용자 데이터 유형을 수집하거나 공유하나요?" → **아니요**
3. **데이터 보안 관행**:
   - "사용자 데이터가 암호화되나요?" → 해당 없음 (데이터 수집 안 함)
   - "사용자가 데이터 삭제를 요청할 방법을 제공하나요?" → 해당 없음
4. 개인정보처리방침 URL 입력

**경로**: 프로덕션 > 국가/지역

**권장 국가** (1단계):

또는 "모든 국가" 선택 가능

---

### 5.2 새 릴리스 만들기
**경로**: 프로덕션 > 릴리스

1. "새 릴리스 만들기" 클릭
2. **App Bundle 업로드**:
   - 파일: `build.aab` (다운로드 링크에서 받은 파일)
   - 버전 코드: 1
   - 버전 이름: 1.0.0
3. **릴리스 이름**:
   ```
   1.0.0 (Initial Release)
   ```
4. **릴리스 노트** (한국어):
   ```
   🎉 NOKK 첫 출시!
   
   주요 기능:
   • 다양한 톤의 안전 사운드 (밝은, 차분한, 강한)
   • 4개 언어 지원 (한국어, English, 日本語, 中文)
   • 다크 모드 지원
   • 오프라인 사용 가능
   • 개인정보 수집 없음
   
   안전한 귀가를 함께합니다! 🛡️
   ```
5. **릴리스 노트** (영어):
   ```
   🎉 NOKK Initial Release!
   
   Features:
   • Safety sounds with various tones (Bright, Calm, Strong)
   • 4 language support (Korean, English, Japanese, Chinese)
   • Dark mode support
   • Offline usage
   • No personal data collection
   
   Your companion for safe walking home! 🛡️
   ```

## ✅ 승인 후 할 일

### 1. URL 확인
```
https://play.google.com/store/apps/details?id=com.nokk.app
```

### 2. 모니터링
- Console에서 설치 수, 리뷰 확인
- 충돌 보고서 모니터링

### 3. 업데이트 준비
버전 업데이트 시:
```bash
# app.json에서 버전 수정
"version": "1.0.1"

# Android versionCode 자동 증가 (EAS가 처리)
npx eas-cli build --platform android --profile production
```

## 🐛 일반적인 거부 사유 및 해결

### 1. 개인정보처리방침 링크 오류
- **문제**: URL이 404 또는 접근 불가
- **해결**: GitHub Pages 활성화 확인

### 2. 스크린샷 문제
- **문제**: 실제 앱과 다름, 저품질
- **해결**: 실제 앱 화면 재촬영

### 3. 설명 불일치
- **문제**: 설명과 실제 기능 불일치
- **해결**: 설명 수정

### 4. 아이콘 문제
- **문제**: 저품질, 저작권 침해
- **해결**: 고품질 아이콘 재작성
