# Screenshot Guidelines for NOKK

## Required Screenshots

### iOS App Store
Create screenshots for these device sizes:
1. **iPhone 6.7" Display** (iPhone 14 Pro Max) - 1290 x 2796 px
2. **iPhone 6.5" Display** (iPhone 11 Pro Max) - 1284 x 2778 px
3. **iPhone 5.5" Display** (iPhone 8 Plus) - 1242 x 2208 px

### Google Play Store
1. **Phone** - Minimum 1080 x 1920 px (16:9 ratio)
2. **Feature Graphic** - 1024 x 500 px (required)

## Screenshot Scenes

Create these 5-6 screenshots showing key features:

### Screenshot 1: Home Screen - Quick Actions
- Show the home screen with 3 quick action buttons
- Highlight: "One Tap = Instant Voice"
- Dark mode preferred

### Screenshot 2: Playing Audio
- Show a quick action button in "playing" state
- Visual feedback showing audio is playing
- Caption: "Instant playback. No delays."

### Screenshot 3: Categories Expanded
- Show accordion-style category list
- One category expanded showing phrases
- Caption: "30+ Ready-to-use phrases"

### Screenshot 4: Premium Features
- Show premium screen with benefits
- Highlight: All phrases, voice tones, custom buttons
- Caption: "Unlock everything with Premium"

### Screenshot 5: Settings - Language
- Show language selection screen
- Display all 4 languages (EN, KO, JA, ES)
- Caption: "Available in 4 languages"

### Screenshot 6: Voice Tone Selection (Optional)
- Show tone selection (Normal, Firm, Angry)
- Caption: "Choose your voice tone"

## Design Guidelines

### Style
- Use dark mode (matches app default)
- Clean, minimal, modern
- Spotify-inspired aesthetic
- Use device frames (mockup)

### Text Overlays
- Short, impactful headlines
- Font: Inter or SF Pro
- Color: White text on dark background
- Primary accent: #1DB954 (green)
- Premium accent: #FFD700 (gold)

### Background
- Solid dark: #121212
- Or subtle gradient

## Tools for Creating Screenshots

1. **Figma** - Design mockups
2. **Sketch** - Design mockups
3. **AppMockUp** - Add device frames (https://app-mockup.com)
4. **Screenshots Pro** - Generate all sizes
5. **Canva** - Quick designs with templates

## File Naming Convention

```
nokk_screenshot_{number}_{device}_{language}.png

Examples:
- nokk_screenshot_01_iphone67_en.png
- nokk_screenshot_02_iphone65_ko.png
- nokk_screenshot_01_android_en.png
```

## Localization

Create localized screenshots for:

Translate headline text overlays for each language.
